# AWS Quiz

It's an AWS simulated test app that I used to study for official exams.

### Features
* [x] Dynamic JSON content
* [x] PWA with mobile support
* [x] Speech Synthesizer 
* [x] Printable result
* [ ] Timer
* [ ] Review flag
